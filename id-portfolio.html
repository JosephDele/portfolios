Project Details

iDiscovery is a mobile app that helps users to enter the details of different activities/events they have participated in. The app has several features which helps users to store different activities that has been entered, activities can be viewed, searched for in case of several entries, updated and deleted. This app has been designed generally to meet the usersâ€™ needs to storing different event details. This application is users centred, general UI principles has been followed and these principles are: consistency, easy reversal of action, recognize rather than recall, simple to use, on touch screen design for thumbs, security and maintainability.
